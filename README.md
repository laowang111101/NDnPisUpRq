# 体检预约后台管理系统 java455

## 项目概述

体检预约后台管理系统是一个基于Vue与Nodejs的前后端分离项目，致力于提供高效、易用的体检预约流程管理。项目采用Vue elementUI进行前端开发，Nodejs作为后端服务，并使用MySQL8.0作为数据库支持。

## 技术栈

- **前端**：Vue + elementUI
- **后端**：Nodejs
- **数据库**：MySQL 8.0
- **Nodejs版本**：14/16/18

## 系统角色

系统定义了以下两种角色：

- **用户**：可以查看体检项目，进行预约等操作。
- **管理员**：负责系统管理，项目管理，以及用户记录管理等。

## 功能模块

以下是系统的核心功能模块：

1. **登录**：提供用户与管理员登录功能。
2. **系统管理**：
   - 系统设置
   - 权限管理
   - 角色分配
3. **项目管理**：
   - 体检项目管理
   - 体检套餐配置
4. **套餐管理**：
   - 套餐信息维护
5. **记录管理**：
   - 预约记录查询
   - 预约状态更新
6. **日志管理**：
   - 用户操作日志
   - 系统运行日志
7. **个人管理**：
   - 个人资料维护
   - 密码更改
8. **轮播图管理**：
   - 首页轮播图配置
   - 轮播内容更新

## 运行环境

- Nodejs环境（14/16/18版本）
- MySQL数据库环境

- **前后端分离**：基于Vue的前端和Nodejs后端分离开发，易于维护与扩展。
- **高效管理**：系统管理模块提供了全面的权限与角色管理，确保系统安全高效。
